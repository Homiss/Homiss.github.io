用Java自建个简易的网页版新浪微博

发表于 2015-05-31   |   分类于 Java三方平台   |  

之所以做这个，只是因为逛到廖雪峰的网站，看见他用Python实现的迷你微博：
http://www.liaoxuefeng.com/article/0013738933956196820a532035641329a1af22170cefc2c000
觉得挺有意思的，所以将他的前台页面扒下来，自己用Java实现后台。

发布微博：

准备工作

网站接入需要在新浪进行认证：http://open.weibo.com/development

如何申请网站认证：http://bbs.lezhizhe.net/forum.php?mod=viewthread&tid=141

对于境外网站，需要我们提供证明，我们可以直接在这里生成认证：http://domaincert.ameyoru.com/

接下来，我们需要在新浪下载Java SDK ，下载地址：http://open.weibo.com/wiki/SDK

下载下来的项目可以直接导入myeclipse中。

开整

找到src文件夹下的config.properties文件，修改

运行example包下weibo4j.examples.oauth2的OAuth4Code类，会弹出微博认证窗口，填上微博账号密码，会跳转至你指定的网站。

将地址栏上code的值复制到

不出意外的话会显示

可以看见你的accessToken的值，拿着这个你就可以对你的微博进行操作了。

出意外的话，直接谷歌、百度。

在SDK已经提供了很多对微博的操作，比如我们需要读取当前账号最新的微博列表，只需要调用其对应的方法即可

更多的功能可以查看文档或者直接看SDK中的weibo4j包下的类。
