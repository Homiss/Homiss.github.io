使用其提供的接口，需要成为UU163合作商户，获得UU163旅行在线为您分配的商户编号、安全验证码，这里就不多说了。

接口开发约定

1）所有的请求和响应数据编码皆为utf-8格式；
2）请使用HTTP POST方法调用接口；
3）JSON中所有字段名称区分大小写；
4）Response内的公共部分包含错误信息结构，因此任何客户端调用在收到reponse消息后需要判断错误信息是否为空，不为空时应根据信息来进行错误处理。
