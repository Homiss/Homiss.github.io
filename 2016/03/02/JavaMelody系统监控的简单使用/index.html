JavaMelody系统监控的简单使用

发表于 2016-03-02   |   分类于 Java 系统监控   |  

JavaMelody主要作用

JavaMelody能够在实际运行生产环境监测Java或Java EE应用程序服务器。并以图表的形式显示：Java内存和Java CPU使用情况，用户Session数量，JDBC连接数，和http请求、sql请求、jsp页面与业务接口方法（EJB3、Spring、Guice）的执行数量，平均执行时间，错误百分比等。图表可以按天，周，月，年或自定义时间段查看。

JavaMelody监控使用很简单：

1.下载相应的jar包

jrobin-x.jar:https://github.com/javamelody/javamelody/releases/download/javamelody-core-1.49.0/jrobin-1.5.9.jar

3.下载javamelody.war放到tomcat/webapps下

5.最后添加好需要监控的web应用就可以了

获取不同格式的数据

文档:http://localhost/javamelody/?format=json

除了监控页面以外，JavaMelody还支持返回XML、JSON等数据格式，例如我们需要返回JSON格式，只需要在链接中加入参数 “?format=json” 即可
