离职了，因为无聊

发表于 2017-09-07   |   分类于 工作   |  

上个月初提了辞职申请，这个月正式离职。

在公司待了一年半的时间，认识了很多有趣的人，接触了很多新的事物，期间也有一些同事离职，有不舍，更多的是祝福。这次终于轮到我，心情还是有些复杂。

不时有朋友问我为啥离职，是太累了还是钱给的不够？

累倒不至于，互联网行业加班是普遍的事，选择这个行业肯定有加班的心理准备，况且自己还年轻，得多拼一把。

至于钱，这个必须承认，肯定有这方面的原因，但不是主要原因。对于刚出社会不久的我来说，这个阶段要说挣多少多少钱，那不是太现实。

离职原因，最主要的，因为无聊。

无聊有两种——外在的无聊和内在的无聊。

外在的无聊，开发趋于稳定，最近一段时间没有新的需求；
内在的无聊，就是工作无挑战，简单说，当工作需要的智力远远低于劳动者的智力时，无聊感就爆发了。当然我这里不是在夸自己，只是因为工作这么久，日常工作大致相同，所以不需要太费脑力。

这人一闲就爱胡思乱想，慢慢我开始变得焦虑，对未来发展的焦虑。

在公司待了一年半，整个公司的项目基本都熟悉，这里对我来说就是个舒适区，舒适区不代表舒适，相反，带给我的更多是焦虑和厌倦，学不到新东西，个人技术没有提升，如果继续待在这，也只能是个最底层码农。

虽然常常调侃自己是条咸鱼，那也是只不甘平凡的咸鱼，所以我选择离职。

离职之后面临最大的问题就是找工作，不得不说找工作真是见压力很大的事情，特别是对于我这种下家都没找好就离职的人。

庆幸的是，在离职前有过一段时间的面试准备，参加了两个公司的面试，都顺利拿到了offer，后来问了下两家公司使用的技术栈，选择了一个更适合自己的公司。

最后，简单说下这次找工作自己得到的经验教训：

  1. 不要离职了再找工作！不要离职了再找工作！不要离职了再找工作！压力太大。
  2. 不断学习新的技术！比如很多招聘都有分布式微服务相关的要求，之前没接触过，所以面试比较吃亏。
  3. 工作中还是需要多做总结，不然面试官问项目中的问题可能答的不会太好。
  4. 英语很重要！英语很重要！英语很重要！
